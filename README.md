❓ Frequently Asked Questions
What is this project?
This is a mood-based song suggester that plays songs depending on your mood.

How does it suggest songs?
You select your mood, and it loads a playlist with matching songs and background animations.

Why did we build this?
To create a fun, interactive way to connect music with emotions for our hackathon challenge.

How does the camera mood detection work?
The camera analyzes facial expressions to detect emotions. For demonstration, we're using a simulated detection, but in a real implementation, this would use AI/ML models.
